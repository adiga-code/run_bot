# Дерево бота — 28-дневная программа бега

## ТОЧКИ ВХОДА

```
/start
  ├─ [Не в вайтлисте] → Кнопка "Подать заявку"
  │     └─ Ввод имени → Заявка уходит к тренеру
  │           ├─ Тренер одобряет → Онбординг
  │           └─ Тренер отклоняет → Уведомление
  │
  ├─ [Онбординг не завершён] → ОНБОРДИНГ (блок 1)
  │
  └─ [Активный пользователь] → Главное меню
```

---

## ОНБОРДИНГ (8 блоков)

### Блок 1: Профиль
```
Фамилия → Имя → Отчество (пропускается) → Пол → Дата рождения
  → Страна → Город → Район (пропускается) → Часовой пояс
```

### Блок 2: Цель
```
Цель (6 вариантов):
  ├─ Начать с нуля
  ├─ Вернуться к бегу
  ├─ Пробежать дистанцию → Какую дистанцию? → Дата старта (пропускается)
  ├─ Улучшить результат
  ├─ Бегать без боли
  └─ Укрепить здоровье
```

### Блок 3: Уровень бега сейчас
```
Бегаете сейчас?
  ├─ Нет → [автозаполнение полей, переход к блоку 4]
  ├─ Нерегулярно → Частота → Объём → Длинная пробежка → Структура
  └─ Регулярно  → Частота → Объём → Длинная пробежка → Структура
```

### Блок 4: Опыт
```
Беговой опыт → Перерыв (если бегали раньше)
```

### Блок 5: Ощущение от бега
```
(Только если бегает) → Как ощущается бег? (легко / средне / тяжело)
```

### Блок 6: Боль
```
Есть боли?
  ├─ Нет → [блок 7]
  └─ Да / немного → Где болит (мультивыбор)
                      → Боль усиливается при нагрузке? → История травм
```

### Блок 7: Другой спорт
```
Другой спорт (мультивыбор) → Частота силовых → Где тренируетесь (зал / дома)
```

### Блок 8: Самооценка → Завершение
```
Самооценка уровня (начинающий / базовый / средний / продвинутый)
  → Алгоритм assign_level() → Уровень 1–5
  → Данные уходят тренеру на одобрение
  → [Ожидание] → Тренер ставит уровень и дату старта
  → Главное меню
```

## УТРЕННИЙ ЧЕК-ИН (кнопка "Сегодняшняя тренировка")

```
Старт чек-ина
  │
  ├─ [Вчерашняя тренировка не отмечена] → Отметить вчера? (да/нет)
  │
  ▼
Самочувствие (1–5)
  → Сон (1–3)
  → Боль (1–3)
       ├─ Боль = 3 или усиливается → КРАСНЫЙ ФЛАГ
       └─ Боль > 0 → Боль усиливается при нагрузке? (да/нет)
  → Стресс (1–3)
  │
  ▼
Движок правил (decide_workout_version)
  ├─ Стресс, боль, сон, самочувствие → счёт
  ├─ Красный флаг? → recovery
  ├─ Накопленная усталость (2+ тяжёлых дня)? → light/recovery
  └─ Итог: base / light / recovery / rest
  │
  ▼
  ├─ [Обычный пользователь] → Ожидание одобрения тренера
  │     └─ Тренер видит карточку → Одобряет / меняет версию
  │           → Тренировка отправляется пользователю
  │
  └─ [Тренер/Админ] → Тренировка отправляется сразу
```

---

## ВЕЧЕРНИЙ ОТЧЁТ (кнопка "Отметить тренировку")

```
Как прошла тренировка?
  ├─ Пропустил → Сохранить (статус: skipped)
  └─ Выполнил / Частично → Усилие (1–5) → Была ли боль? (да/нет)
                                              → Сохранить (done / partial)
  │
  ▼
  ├─ [День 28] → Поздравление с завершением программы
  └─ [День 7/14/21 и завершённость < 75%] → Неделя повторяется
```

---

## МОЙ ПРОГРЕСС

```
Показатели:
  - Уровень (1–5): Начальный / Возвращение / Базовый / Стабильность / Результат
  - Календарный день (1–28)
  - Шаблонный день (повторяется при <75% завершённости)
  - Завершённость текущей недели (%)
  - Всего тренировок выполнено
  - Стрик (дней подряд)
  - Тип тренировок на ближайшие 7 дней
```

---

## НАПОМИНАНИЯ

```
Меню напоминаний
  ├─ Включить / выключить
  ├─ Утреннее время (5–23 ч) → Выбрать час
  └─ Вечернее время (5–23 ч)  → Выбрать час
```

---

## ЛОГИКА ПОВТОРА НЕДЕЛИ (прогресс)

```
После дня 7 / 14 / 21:
  - Завершённость >= 75% → следующая неделя (уровень может вырасти)
  - Завершённость <  75% → текущая неделя повторяется
После дня 28:
  - [Опционально] 5-я неделя для продвинутых
```

---

## ДВИЖОК ПРАВИЛ — ВЕРСИИ ТРЕНИРОВОК

| Версия    | Когда назначается                                    |
|-----------|------------------------------------------------------|
| `base`    | Нормальное состояние                                 |
| `light`   | Небольшая усталость / плохой сон / стресс            |
| `recovery`| Высокая боль / красный флаг / сильная усталость     |
| `rest`    | День отдыха по расписанию (независимо от чек-ина)   |

## АДМИН-ПАНЕЛЬ

```
/admin
  ├─ Ожидающие одобрения (новые пользователи)
  │     └─ Выбрать уровень → Дата старта (сегодня/завтра)
  │
  ├─ Все пользователи
  │     └─ Профиль пользователя
  │           ├─ Изменить уровень
  │           ├─ Изменить дату старта
  │           ├─ Поставить статус (активен/пауза/ожидание)
  │           ├─ Перейти на день N
  │           ├─ Отметить день вручную
  │           ├─ Расширить до 5-й недели
  │           ├─ Отправить сообщение
  │           └─ Сбросить (вернуть в онбординг)
  │
  ├─ Одобрение чек-инов
  │     └─ Карточка с данными → base / light / recovery / rest → Отправить тренировку
  │
  ├─ Аналитика
  │     ├─ KPI (всего/активных/ожидающих, % завершённости за 7 дней)
  │     ├─ График завершённости за 14–90 дней
  │     └─ Таблица по уровням
  │
  └─ Рассылка напоминаний
```

## ТИПЫ ТРЕНИРОВОК (по дням)

| Тип        | Описание                        |
|------------|---------------------------------|
| `run`      | Беговая тренировка              |
| `strength` | Силовая (зал или дома)          |
| `recovery` | Восстановление / растяжка       |
| `rest`     | Полный день отдыха              |

Каждый тип имеет версии: `base`, `light`, `recovery`

Уровни: 1 (Начальный) → 2 (Возвращение) → 3 (Базовый) → 4 (Стабильность) → 5 (Результат)

## ФОНОВЫЕ ЗАДАЧИ (Scheduler)

```
Утро (по часовому поясу пользователя):
  → Напоминание о чек-ине (если включены напоминания и ещё не выполнен)

Вечер (по часовому поясу пользователя):
  → Напоминание отметить тренировку (если не отмечена)
```
